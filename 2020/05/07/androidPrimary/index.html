Android初级学习路线

第一关：四大组件（建议学习时间：一周）

知识点：四大组件的使用、Activity启动模式、Service的start和bind

、Activity启动模式、Service的start和bind

1. Android 四大组件 - 简书

2. 彻底弄懂Activity四大启动模式 - Android研发专栏 - CSDN博客

第二关：布局和控件（建议学习时间：3天）

知识点：常见的布局和布局参数

1.Android系统五大布局详解Layout - 漫步云端 - CSDN博客

2. Android 基础：常用布局 介绍 & 使用（附 属性查询） - 简书

第三关：简单自定义View（建议学习时间：一周）

知识点：会简单的自定义View即可。在入门阶段，不适合了解measure、layout和draw过程，不建议去学习复杂的自定义View。

1. 自定义View，有这一篇就够了 - 简书

2. GitHub - chenzongwen/MiClockView

3. Android 自定义 View 合集 - Android - 掘金

第四关：动画和手势（建议学习时间：2周）

要求：掌握View动画、属性动画、布局动画、手势

1. Android中的View动画和属性动画 - 简书

2. HenCoder Android 自定义 View 1-6：属性动画 Property Animat…

3. Android属性动画深入分析：让你成为动画牛人 - 任玉刚 - CSDN博客

4. android 动画系列 (3) - layoutAnimation 视图动画 - 简书

5. RecyclerView 与 LayoutAnimation 实现的进入动画（一 ）: List -…

6. 安卓自定义View进阶-手势检测(GestureDetector)

7. 安卓自定义View进阶-缩放手势检测(ScaleGestureDecetor)

第五关：网络（建议学习时间：1周）

要求：了解HTTP/HTTPS、状态码、header，get和post等；掌握网络请求API和相关网络库；熟悉长连接。

1. HTTP 必知必会的那些

2. Android HttpURLConnection详解 - 简书

3. 手把手教你写 Socket 长连接

第六关：图片加载（建议学习时间：2周）

要求：了解drawable加载规则以及图片缓存

1. Android中Bitmap内存优化 - 简书

2. Android drawable微技巧，你所不知道的drawable的那些细节 - 郭霖的专栏

3. Android照片墙完整版，完美结合LruCache和DiskLruCache - 郭霖的专栏

4. Android DiskLruCache完全解析，硬盘缓存的最佳方案 - 郭霖的专栏 - CSDN博…

第七关：文件和数据库 (建议学习时间：1周)

要求：了解文件流、sqlite

1. Java 随机访问文件_w3cschool

2. Java 序列化的高级认识

3. Android SQLite详解 - 简书

4. Concurrent database access - Dmytro Danylyk

第八关：Handler (建议学习时间：3天)

要求：会使用Handler，知道looper的概念 学习资料：

1. Android Handler的基本使用 - 简书

2. Android Handler详解 - 简书

第九关：异步线程池 (建议学习时间：7天)

要求：熟练使用线程、线程池、线程同步

1. Android 线程和线程池一篇就够了 - Android - 掘金

2. Java线程同步：synchronized锁住的是代码还是对象 - 叉叉哥的BLOG - CSDN博…

3. Java中的wait、notify、notifyAll - 简书

4. Java并发编程中级篇(三)：使用CountDownLatch控制多线程并发等待 - 阿拉德大陆的魔…

第十关：Resources (建议学习时间：1天)

要求：掌握Android中访问资源的方式，好吧，这一关有点简单

1. Android xml资源文件中@、@android:type、@*、？、@ 含义和区别 - 小M…

2. Android根据资源名获取资源ID - 技术小黑屋

第十一关：Retrofit / Okhttp (建议学习时间：3天)

要求：会使用Retrofit / Okhttp进行日常开发，复杂功能可以通过搜索和查阅官方文档解决即可

1. Retrofit RxJava ＋ OkHttp 让网络请求变的简单-基础篇 - 简书

2. Retrofit RxJava ＋ OkHttp 让网络请求变的简单-封装篇 - 简书

3. Retrofit

要求：会使用rxjava进行日常开发，复杂功能可以通过搜索和查阅官方文档解决即可
